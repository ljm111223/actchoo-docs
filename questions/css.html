# CSS 选择器

# 为什么把 CSS 文件放在 <head> 中，而把 JS 文件放在 <body> 底部

  1. CSS 会阻塞页面渲染，越早加载越好
  2. JS 会阻塞 DOM 树的生成以及渲染，越晚加载越好

# CSS 中伪类与伪元素的区别

# 伪类

核心是用来选择 DOM 树之外的信息，不能够被普通选择器选择的文档之外的元素，用来添加一些选择器的特殊效果。

由于状态的变化是非线性的，所以元素达到一个特定的状态时，它可能得到一个伪类的样式，单当状态改变时，它又会失去这个样式。

由此可以看出，它的功能和 class 有点类似，但它是基于文档之外的抽象，所以叫伪类。

# 类型

# 伪元素

DOM 树没有定义的虚拟元素，核心就是创建通常不存在于文档中的元素。

伪元素控制的内容和元素是没有区别的，但是它本身只是基于元素的抽象，并不存在于文档中，所以称为伪元素，用于将特殊的效果添加到某些选择器。

# 成员
  1. ::before 、 ::after
  2. ::selection

# 区别

# 表示方法
  1. CSS2.1 之前伪类和伪元素都是单冒号 : 表示
  2. CSS2.1 及之后规定伪类用单冒号: 表示，伪元素用 :: 表示
  3. 浏览器也支持之前的写法，但应该按照建议规范来
# 定义
  1. 伪类即假的类，是基于普通 DOM 元素⽽产⽣的不同状态，他是 DOM 元素的某⼀特征
  2. 伪元素即假元素，创建在 DOM 树中不存在的抽象对象，⽽且这些抽象对象是能够访问到的

# More：相同

伪类和伪元素都不出现在源⽂件和 DOM 树中，也就是说在 html 源⽂件中是看不到伪类和伪元素的。

# CSS 属性

# 控制继承的属性： inherit、initial 和 unset 三者的区别

# initial

initial 关键字用于设置 CSS 属性为他的默认值，可用作于任何 CSS 样式。

简单来说，就是初始化到该属性浏览器默认定义的值。

# inherit

每一个 CSS 属性都有一个特性，就是这个属性必然是默认继承（inherited: yes）或默认不继承（inherited: no）

# 属性的继承特性
  1. 所有元素可继承：visibility 和 cursor
  2. 内联元素可继承：letter-spacing 、word-spacing、white-space、line-height、color、font、font-family、font-size、font-style、font-variant、font-weight、text-decoration、text-transform、direction
  3. 块级元素可继承：text-indent 和 text-align
  4. 列表元素可继承：list-style、list-style-type、list-style-position、list-style-image
  5. 表格元素可继承：border-collapse

例如，border 是不可继承的，可以通过 inherit 来进行改变：

.box {
    width: 100px;
    height: 100px;
    border: 1px solid red;
}

.child {
    width: 40px;
    height: 20px;
    /* border 就继承了父级 .box 的 border 样式 */
    border: inherit;
}

# unset

unset 关键字可以理解为不设置。其实，它是关键字 initial 和 inherit 的组合。

设置 unset 会有以下几种情况：

  1. 如果该属性是默认继承属性，该值等同于 inherit
  2. 如果该属性是非默认继承属性，那么该值等同于 initial
  3. 换句话说，unset 关键字会优先用 inherit 关键字的样式，其次使用 initial 关键字的样式
